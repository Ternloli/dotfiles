# dotfiles

My personal configuration files for Linux/macOS development environment.

## Contents

- `.zshrc` - Zsh configuration with Oh My Zsh
- `.vimrc` - Vim configuration
- `.gitconfig` - Git global configuration
- `tmux.conf` - Tmux configuration
- `starship.toml` - Starship prompt configuration

## Installation

```bash
git clone https://github.com/Ternloli/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Requirements

- zsh
- vim
- tmux
- starship
